# Индексы в PostgreSQL

## Описание проекта

В этом проекте используются:

- Создание различных типов **индексов** для оптимизации запросов.
- Использование **EXPLAIN ANALYZE** для анализа работы индексов.
- Создание уникальных и частичных индексов.
- Применение мульти-колоночных индексов для ускорения запросов.
