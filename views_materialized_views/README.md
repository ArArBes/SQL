# Виртуальные и материализованные представления

## Описание проекта

В этом проекте используются:

- Создание **Database Views** для выборки данных с фильтрацией.
- Использование **Materialized Views** для сохранения данных с вычислениями.
- Применение операций с датами и вычислениями в представлениях.
- Работа с обновлением и удалением данных в представлениях.
