# Персональные скидки

## Описание проекта

В этом проекте используются:

* Создание таблицы **person_discounts** для хранения информации о персональных скидках пользователей в пиццериях.
* Заполнение таблицы с использованием **INSERT INTO ... SELECT** и агрегации данных.
* Создание уникального многоколонкового индекса **idx_person_discounts_unique** для предотвращения дублирования записей.
* Добавление **ограничений целостности данных** для столбцов `person_id`, `pizzeria_id` и `discount`.
* Создание **последовательности** для автоматической генерации значений первичных ключей.
