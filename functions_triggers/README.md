# Работа с функциями и триггерами в PostgreSQL

## Описание проекта

- **Аудит данных**: создание триггеров для отслеживания вставок, обновлений и удалений данных в таблицах.
- **Работа с функциями**: создание SQL и PL/pgSQL функций для выполнения различных операций, включая работу с параметризованными функциями и функциями-обертками.
- **Оптимизация запросов**: использование функций для упрощения запросов и улучшения их производительности.
