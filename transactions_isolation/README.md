# Работа с транзакциями и уровнями изоляции в PostgreSQL

## Описание проекта

- **Транзакции в PostgreSQL** для обеспечения атомарности операций.
- **Уровни изоляции транзакций** для управления параллельным доступом к данным.
- **Репродукция классических аномалий данных** для анализа их поведения в различных уровнях изоляции.
