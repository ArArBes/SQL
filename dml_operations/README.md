# DML

## Описание проекта

В этом проекте используются:

- Изменение данных с помощью **INSERT**, **UPDATE**, **DELETE**.
- Работа с операциями **UNION** и **JOIN** для выборки и обработки данных.
- Регистрация новых пицц, заказов и посещений.
- Актуализация цен и данных в базе.
- Удаление данных на основе условий.
