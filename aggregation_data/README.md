# Aggregation data

## Описание проекта

В этом проекте используются:

- **Агрегация данных** с вычислением сумм, средних значений, минимальных и максимальных значений.
- **Группировка данных** по атрибутам и фильтрация результатов.
- **Объединение таблиц** с использованием различных SQL операторов.
- **Работа с условной логикой** для вычисления бизнес-метрик и отчетности.
- **Применение агрегатных функций** для анализа данных о клиентах и ресторанах.
