# SQL

Проект включает практические навыки работы с SQL. 

## Задачи проекта

- **Создание и настройка таблиц**: Разработка таблиц с ключами, ограничениями и типами данных, а также манипуляции с данными через DDL и DML запросы.
- **Оптимизация производительности**: Использование индексов, представлений и агрегированных запросов для повышения скорости работы базы данных.
- **Автоматизация через функции и триггеры**: Создание и настройка функций и триггеров для обработки запросов и автоматизации операций с данными.
- **Аудит изменений**: Реализация механизма отслеживания вставок, обновлений и удалений данных в таблицах.
- **Управление транзакциями**: Настройка транзакций, уровней изоляции и решение проблем с конкурентным доступом.
- **Работа с материализованными представлениями**: Использование представлений для упрощения запросов и ускорения обработки данных.

## Описание проекта

Проект состоит из нескольких упражнений, демонстрирующие навыки работы с реляционными базами данных:

- Проектирование таблиц с ключами и ограничениями.
- Настройка транзакций и уровней изоляции для обеспечения целостности данных.
- Аудит изменений с использованием триггеров и функций.
- Оптимизация запросов и повышение производительности с помощью индексов и представлений.
- Разработка бизнес-логики с использованием PL/pgSQL для реализации сложных операций.
- Создание аналитических отчетов и агрегированных данных.

## Технологии

- **PostgreSQL**
- **SQL**
- **PL/pgSQL**
